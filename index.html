📁 আমার প্রজেক্ট 📞 যোগাযোগ

👤 আমার সম্পর্কে

কিছু কথা আমার সম্পর্কে

আমি একজন প্যাশনেট ওয়েব ডেভেলপার। HTML, CSS, JavaScript শিখে নিজের ওয়েবসাইট নিজেই বানিয়েছি।

এখন পর্যন্ত ১০+ প্রজেক্ট তৈরি করেছি—ডায়েরি অ্যাপ, গেম, ক্যালকুলেটর, টু-ডু লিস্ট সহ আরও অনেক কিছু।

আমার লক্ষ্য একজন ফুল-স্ট্যাক ডেভেলপার হওয়া এবং পরিবারের পাশে দাঁড়ানো।

💪 আমার স্কিল

যেসব টেকনোলজি নিয়ে কাজ করছি

HTML

95%

HTML (HyperText Markup Language) দিয়ে ওয়েবসাইটের মৌলিক গঠন তৈরি করি। টেবিল, লিস্ট, ছবি, লিংক সবই HTML দিয়ে বানানো।

CSS

85%

CSS দিয়ে ওয়েবসাইট সুন্দর ও রঙিন করি। অ্যানিমেশন, গ্রেডিয়েন্ট, ফ্লেক্সবক্স, গ্রিড সবই শিখেছি।

JavaScript

70%

JavaScript দিয়ে ওয়েবসাইটে প্রাণ দিই। DOM ম্যানিপুলেশন, ইভেন্ট হ্যান্ডলিং, API, লোকাল স্টোরেজ—সবই পারি।

Git & GitHub

60%

Git দিয়ে কোড ভার্সন কন্ট্রোল করি, আর GitHub-এ প্রজেক্ট আপলোড করে লাইভ সাইট বানাই।

📁 আমার প্রজেক্ট

এ পর্যন্ত যেসব প্রজেক্ট বানিয়েছি

📓 ডায়েরি অ্যাপ

নোট যোগ, এডিট, ডিলিট, সার্চ, পিন—সব ফিচার সহ প্রফেশনাল ডায়েরি।

🎮 পাথর-কাগজ-কাঁচি

কাউন্টডাউন, সাউন্ড, কনফেট্টি, কম্বো সিস্টেম সহ প্রো গেম।

🧮 ক্যালকুলেটর

সুন্দর UI, কীবোর্ড সাপোর্ট, হিস্টোরি সহ পূর্ণাঙ্গ ক্যালকুলেটর।

✅ টু-ডু লিস্ট

টাস্ক যোগ, এডিট, ডিলিট, ফিল্টার, প্রোগ্রেস বার, ড্র্যাগ & ড্রপ।

🌐 পোর্টফোলিও

নিজের ওয়েবসাইট—১২+ পেজ, ডার্ক/লাইট মোড, মাল্টি-ল্যাঙ্গুয়েজ।
